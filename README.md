# student-marks-analysis-system
A Python &amp; NumPy project for analyzing student marks with statistics, grading, topper detection, and pass/fail analysis.
# 🎓 Student Marks Analysis System

A beginner-friendly Python project built with **NumPy** to analyze student marks and generate meaningful insights using statistical functions.

This project demonstrates how NumPy can be used to perform fast numerical computations, analyze student performance, and automate grading.

---

## 📌 Features

- 📊 Store student names and marks using NumPy arrays
- 📈 Calculate Average (Mean)
- 📉 Calculate Median
- 🏆 Find the Topper
- 📌 Find Highest & Lowest Marks
- 📏 Calculate Standard Deviation
- ✅ Identify Passing Students
- ❌ Identify Failing Students
- 📊 Calculate Pass Percentage
- 🎖️ Assign Grades Automatically
- 📋 Display a Complete Student Performance Report

---

```bash

```

3. Install NumPy


4. Run the project

```bash
python Student_Marks_Analysis.py
```
